## 🚀 本地开发

### 环境要求

- Node.js 18+
- pnpm（推荐）或 npm

### 安装与运行

```bash
# 克隆仓库
git clone https://github.com/Awfp1314/awfp1314.github.io.git
cd awfp1314.github.io

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

访问 `http://localhost:4321` 查看博客。

### 创建新文章

```bash
# 使用脚本创建新文章
pnpm new
```

按提示输入文章标题、分类等信息，脚本会自动生成文章模板。

### 构建部署

```bash
# 构建生产版本
pnpm build

# 预览构建结果
pnpm preview
```
