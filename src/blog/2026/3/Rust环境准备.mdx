# Rust环境准备

# 安装

## MacOS/Linux

1、终端输入命令：

```
$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

2、等待安装成功出现以下命令

> Rust is installed now. Great!

## Windows

1、[安装包下载](https://rust-lang.org/tools/install/)
2、安装好后运行安装包，选择第一个默认安装
3、等待安装好之后输入 rustc --version,若出现版本提示则按照成功

# 更新和卸载

## 更新

在终端输入

```
rustup update
```

## 卸载

在终端输入

```
$ rustup self uninstall
```
